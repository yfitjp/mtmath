外角定理

三角形において、一つの外角は、それと隣り合わない二つの内角の和に等しい。数式で表すと、{"type": "formula", "content": "\angle D = \angle A + \angle B"}。ここで、{"type": "formula", "content": "\angle D"}は外角、{"type": "formula", "content": "\angle A"}と{"type": "formula", "content": "\angle B"}は隣り合わない二つの内角を表す。



  • 外角定理は、三角形の一つの外角が、それと隣り合わない二つの内角の和に等しいことを示す。この定理は、三角形の性質を理解し、図形問題を解く上で重要な役割を果たす。


証明

三角形ABCを考え、頂点Cにおける外角を{"type": "formula", "content": "\angle D"}とする。

三角形の内角の和は180度であるため、{"type": "formula", "content": "\angle A + \angle B + \angle C = 180^\circ"}が成り立つ。

外角{"type": "formula", "content": "\angle D"}は、{"type": "formula", "content": "\angle C"}と隣り合う角であるため、{"type": "formula", "content": "\angle D + \angle C = 180^\circ"}が成り立つ。

上記の2つの式から、{"type": "formula", "content": "\angle A + \angle B + \angle C = \angle D + \angle C"}が導かれる。

両辺から{"type": "formula", "content": "\angle C"}を引くと、{"type": "formula", "content": "\angle A + \angle B = \angle D"}が得られる。

したがって、三角形の一つの外角は、それと隣り合わない二つの内角の和に等しい。
